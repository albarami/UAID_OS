## 0. Goal & non-goal
- **Goal.** A **deterministic-gated, LLM-assisted** canonical-artifact generator (Slice‑35 inert model — **NO tool broker**) that turns an **accepted** source document into an **inert, NON-BINDING** §6.3-typed draft in a **new `generated_artifacts` store**, stamped `system_authored_unapproved` (§7.2), with the generator lineage recorded (§7.4). An **independent approval** carrying **bound §7.3 evidence** (for `human_owner` or `independent_agent_lineage`) moves a draft to `system_authored_human_approved`/`system_authored_independent_approved` (binding-eligible authorship) or `disputed` (blocked). One artifact draft per call, parameterized by a **requested** `artifact_type`; generation reuses the Slice‑35 injection-safe / budget-gated / incurred-cost / FakeLLM pipeline.
- **Non-goal.** **STRICTLY STORE/INFRA-ONLY (B2):** NO spine write, NO promotion ⇒ `production_autonomy.py`/`readiness.py` UNTOUCHED, **bit-stable** (`before==after`, `ruleset_version` unchanged, NO A5-gate movement, go-live false). Drafts **never binding before independent approval** (§7.1/§7.2), **never enter the spine** here. **`*_approved` is independence-evidence-recorded, NOT verified** (caller labels unverified — B1). **`domain_authority` + `reference_oracle` approval bases are DEFERRED** (rejected this slice — v3). **No `unknown` artifact_type** (B3). **No denylist/redaction/no-secret guarantee** for stored `title`/`body` (B5). No tool broker / external I/O beyond the LLM call. No HTTP endpoint. Deep per-type schemas, multi-source generation, and spine promotion are deferred. Live provider untested.

### 3.2 §7.2 / §7.3 / §7.4 vocab
`AUTHORSHIP_STATUSES` = the 6 (verbatim); `GENERATED_INSERT_STATUS="system_authored_unapproved"`; `APPROVED_STATUSES=(system_authored_human_approved, system_authored_independent_approved)`; **`APPROVAL_BASES=(human_owner, independent_agent_lineage)`** (the two supported §7.3 paths; `domain_authority`/`reference_oracle` DEFERRED — v3); `OUTCOMES=(succeeded, refused_injection, blocked_by_budget, failed)`; `PROMPT_VERSION="generate.v1"`; `GENERATE_SYSTEM_PROMPT` (untrusted-data framed, STRICT JSON `{title, body}`).
### 3.3 Independence rules (`validate_independence`) + cost
- `disputed`: status only (no evidence).
- `system_authored_human_approved`: `approval_basis='human_owner'` + `approved_by` (non-empty, **`<> generated_by`**) + `approved_at` + `reviewer_authority` (the human owner's authority). (Prompt-family independence not required for a human owner.)
- `system_authored_independent_approved`: `approval_basis='independent_agent_lineage'` + `approved_by` (**`<> generated_by`**) + `approved_at` + **`reviewer_prompt_family <> generator_prompt_family`** + `reviewer_role` + `reviewer_authority` (separate reviewer authority, §7.3).
- **Any other `approval_basis` (`domain_authority`, `reference_oracle`, or unknown) ⇒ refused** (deferred; fail-closed). `validate_authorship_transition` = one-way `system_authored_unapproved → {*_approved, disputed}`.
- **Cost = incurred-cost, reused verbatim from Slice‑35:** no cost for injection-refuse/budget-block/exception/invalid-token; cost on any valid-token response BEFORE parse (parse failure keeps `cost_external_ref`+tokens). `parse_generated_artifact(raw)→GeneratedDraft(title, body)`.
