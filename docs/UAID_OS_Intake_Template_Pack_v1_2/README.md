# UAID OS Universal Intake Template Pack v1.2

This pack contains the 26 canonical intake files required for R5 autonomous go-live readiness. Projects may submit different formats, but UAID OS should compile them into this canonical structure.

The pack is domain-agnostic. Domain-specific requirements belong in `10_domain_pack.yaml`, `09_test_oracles.yaml`, and any domain authority records, not in the core platform specification.
